字体与排版

标题字体：Noto Sans SC，粗体

行高：1.5 倍字号，正文 ≥ 16px

色彩体系

主色：深蓝 #1E3A8A（专业、稳重）

辅助色：灰色系 #6B7280（中性色）

点睛色：亮蓝 #2563EB 或绿色 #10B981（突出按钮/高亮信息）

遵循 60-30-10 原则

使用 Element Plus 栅格系统 (el-row / el-col)

卡片: el-card，圆角 12px，轻微阴影

按钮: el-button 主按钮深蓝背景，白色文字；悬停时加深或渐变

表格: el-table，斑马纹、hover 高亮

表单: el-form + el-form-item，聚焦时高亮描边

交互与动效

微交互: 按钮点击、表单校验反馈

过渡动画: transition 或 Element Plus 自带动画，200-300ms

避免复杂动画，保持简洁

响应式

使用 el-row 的 :gutter 与 :span 控制不同屏幕布局

移动端单列 → 平板两列 → PC 三列或四列布局

🛠 技术规范

语言: TypeScript

框架: Vue3 + Vite

图标库: Element Plus 内置图标或 Font Awesome
